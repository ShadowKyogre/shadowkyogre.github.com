Mikidown Personal

by ShadowKyogre and rnons

Screenshots

All screenshots are located here

What is this?

It's like the original mikidown, but different. It doesn't have a site generator. Nor does it have whoosh. Or the TOC for the edit window. But it DOES work on Windows. It also uses a QPlainTextEdit in place of a QTextEdit. That's it.

Download


You can also clone the project with Git by running:
$ git clone git://github.com/ShadowKyogre/mikidown-personal
